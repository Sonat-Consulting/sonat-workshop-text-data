



ELMER

Last updated: 5 September 2021.





ELMER (Easier and more efficient reporting) is a set of guidelines for public web forms.

The guidelines will help to ensure that public forms for the private sector are designed in a uniform and user friendly manner. Elmer is the public administration standard for forms on the internet. The Brønnøysund Register Centre administers the Elmer guidelines.
