The data set is provided in an API (Application Programming Interface) that supports both single lookups, complex searches and downloads of complete data sets. You can also search for information for presentation directly on screen.
Licence
The data sets follow the provisions of the Norwegian Licence for Open Government Data (NLOD). It is not necessary to register in order to use the data set.
