Datasettet tilbys i et API (Application Programming Interface) som støtter både enkeltoppslag, sammensatte søk og nedlasting av komplett datasett. Du kan også søke etter informasjon for presentasjon direkte på skjerm.
Lisens
Datasettene følger Norsk lisens for åpne data (NLOD). Det er ikke nødvendig å registrere seg for å ta datasettet i bruk.
